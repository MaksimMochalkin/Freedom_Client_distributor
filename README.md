# Freedom_Client_distributor
Консольное приложение реализующее логику по назначению клиентов на менеджеров исходя из гео-позиции клиента.
Для старта работы с приложением, после запука наберите команду "--help" 
Для парсинга данных из csv используется библиотека CsvHelper.
Для определения гео-позиции используется Yandex geao api. Для его использования необходимо в переменную среды установить ваш api ключ.
Программа по умолчанию использует моковые данные для определения гео-позиции.
Итоговый json с конечными данными по умолчанию сохраняется в папку "Documents" на вашем компьютере с названием файла "assigment_result.json"
